┌─────────────────────────────────────────────────────────────────────────────┐
│                          Performance Notes                                  │
└─────────────────────────────────────────────────────────────────────────────┘

    • Socket latency: ~1ms for local UNIX sockets
    • No overhead: Cross-instance = same-instance spawn time
    • Linear scaling: N instances, no shared state bottlenecks
    • Tested: 4 instances, 14 windows, no degradation
